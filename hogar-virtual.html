✦ Hogar Virtual de Ed
Hola Ed 👋 Vamos a conectar tu base de datos para que puedas abrir tu hogar desde cualquier dispositivo con tu correo. Solo lo haces una sola vez.
📋 PASOS RÁPIDOS (5 minutos)
1
Abre console.firebase.google.com en otra pestaña. Inicia sesión con edespevi@gmail.com
2
Clic en "Agregar proyecto" → Nombre: hogar-ed → Continuar → Continuar → Crear proyecto
3
Menú izquierdo → Build → Authentication → Comenzar → activa Correo/Contraseña → Guardar
4
Menú izquierdo → Build → Realtime Database → Crear base de datos → cualquier región → Modo de prueba → Listo
5
Ícono ⚙️ arriba → Configuración del proyecto → busca "ID del proyecto" y cópialo abajo. También copia la Clave de API web
6
Ve a Realtime Database → copia la URL que aparece arriba (ej: https://hogar-ed-default-rtdb.firebaseio.com)
